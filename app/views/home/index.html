Git is a tool used by developers to manage their projects in a safe manner and it provides an important mechanism to backup code and retrieve lost work.

It also allows multiple developers to work together on the same files at the same time without interfering with each other's work. You can even work offline and update when you get a connection.

Since Git is a "cornerstone" tool, we start off teaching it right away, along with the UNIX Terminal and Vim.

HTML is the backbone of the Interwebs. It is the structure and content element. You are right now reading words wrapped in HTML.

HTML5 makes huge strides in adding functionality to the webs like native video and audio without the use of Flash. There is even a spec to handle video conferencing called WebRTC!

If HTML is the backbone of the interwebs, than CSS is the clothes, hair, and possibly make-up, if you are into that.

But CSS is more than just the color and typeface. It is also responsible for responsive design and element positioning in general. It is an invaluable skill when selling yourself as a new Web Developer.

Javascript is THE language of the Internet. It is the only scripting language native to all browsers. If you want your app to something more than just sit there and look pretty you need Javascript to get a job today in Web Development.

The number one skill today if you want a job in programming is Javascript. More than Ruby, more than Rails, and more than PHP.

Along with interaction, Javascript provides mechanisms to request data from another location on the Internet. You can use it to parse NASA, NASCAR, and NASDAQ data.

Ruby is a server-side language and it is used to work with databases and write complex software in a friendly, readable manner.

In fact, Ruby is a programming language designed for optimizing developer joy! It is easy to pick up and is powerful enough to take years to master.

At ODS we offer advanced courses in Ruby and Ruby on Rails as well!

The main underpinning of all web software is the network protocols that lie underneath. These are all based on standards set by UNIX. UNIX also comes with an amazing "bag of tricks", little, powerful programs that can be "chained" together to create amazing functionality.

Vim is an amazing way to edit text of all kinds, from books, to software code. It is a highly adaptable UNIX Terminal program that can be run locally, or even over a network.
